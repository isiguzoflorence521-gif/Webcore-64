# Webcore 64 ROMs Directory

This folder contains the N64 game files optimized for the **Webcore 64** engine.

### ⚠️ Critical Requirements:
1. **Format**: All ROMs must be in `.zip` format.
2. **Case Sensitivity**: Filenames must match `games.json` exactly (e.g., `sm64.zip`).
3. **Pathing**: The `player.html` uses the relative path: `gameUrl: 'roms/'`.

### Active Library:
* **Banjo-Kazooie** (`banjo_kazooie.zip`)
* **Paper Mario** (`paper_mario.zip`) - *Specialized HLE fix for Pause Menu lag.*
* **Super Mario 64** (`sm64.zip`)
* **Super Smash Bros.** (`smash_bros.zip`)
* **Star Fox 64** (`starfox64.zip`)
* **Wave Race 64** (`wave_race_64.zip`) - *Optimized Threaded Rendering for water physics.*
